# AI-Powered Sustainability Analytics — Interactive Demo

An interactive click-through demo showcasing **Conversational AI Analytics** with Microsoft Fabric Data Agents.

🔗 **[View Live Demo](https://agenticsociety.github.io/sustainability-analytics-demo/)**

## Navigation

- **Click** sidebar items to jump to any beat
- **→** or **Space** to advance, **←** to go back, **Esc** to reset

## Technology

- **Data**: Microsoft Fabric Lakehouse + Data Agents (MCP)
- **AI**: Claude (Anthropic) multi-agent orchestration
- **Viz**: Chart.js + custom HTML/CSS
- **Analytics**: Google Analytics 4 + Microsoft Clarity
