Tier 1 — Available today with current agents
1. Emissions intensity ranking
t CO₂e per €M revenue — identifies which divisions earn most carbon-efficiently
2. Carbon price stress test
Simulate €50–300/t scenarios to quantify margin erosion per division
3. Portfolio priority matrix
Revenue × intensity quadrant for decarbonization capital allocation
4. Green revenue ratio
% of revenue from below-average intensity divisions — ESG KPI
Tier 2 — Unlocked by enriching the Lakehouse
5. Decoupling analysis
Add time series → track if revenue grows faster than emissions (absolute vs. relative decoupling)
6. Scope 1/2/3 waterfall
Add scope breakdown → identify PPA vs. process optimization levers per division
7. Facility-level hotspot map
Add plant/country data → geo-map of emission hotspots vs. revenue contribution per site
8. Product carbon footprint
Add product-group revenue → PCF per bearing, per e-drive, per aftermarket part
Tier 3 — With external data agents
9. CBAM exposure calculator
+ EU tariff rates agent → auto-calculate border adjustment costs by division and trade flow
10. Peer benchmarking
+ industry data agent → compare intensity vs. competitors (SKF, Continental, Nidec, ZF)
11. SBTi pathway tracker
+ SBTi target agent → visualize actual trajectory vs. 1.5°C pathway per division
12. Grid decarbonization ROI
+ energy market agent → model PPA cost vs. carbon cost savings by plant location

The key insight
The AI layer doesn't need to change — it's the data that scales. Every new dimension added to the Fabric Lakehouse (time series, scope breakdown, facility, product group) unlocks exponentially more cross-cutting questions through the same conversational interface. The investment is in AI-ready data, not in new dashboards.
